# 博客前台文档

本目录用于存放 `apps/blog` 的应用内文档。

适合放在这里的内容：

- 前台页面与路由说明
- 前台组件与交互约定
- SEO、PWA、搜索等前台专项说明
- 只对博客前台生效的实现记录

不适合放在这里的内容：

- 仓库级任务清单
- 共享基础层盘点
- 跨应用通用架构说明

这些内容应放回仓库根目录下的 `docs/`。

    - Blog 前台上线前最终优化主任务文件
